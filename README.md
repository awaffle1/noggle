# Noggle

### The NationStates search engine

Noggle is a search engine powered by Google for NationStates. It can search the main page (www.nationstates.net) and the forums (forum.nationstates.net).

You can get support [here](https://web.libera.chat/?channel=#mediawiki).

There is also a [browser](https://github.com/noggle-tech/noggle-browser).
